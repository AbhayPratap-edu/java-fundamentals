**View and manage commit history**

| Command | Purpose | Example |
|---------|---------|---------|
| `git log` | Full commit history | `git log` |
| `git log --oneline` | Compact history | `git log --oneline` |
| `git log --graph --all --decorate` | Visual graph | `git log --graph --all` |
| `git log -n 5` | Last 5 commits | `git log -n 5` |
| `git log --author="John"` | Filter by author | `git log --author="John"` |
| `git log --since="2 weeks ago"` | Filter by date | `git log --since="1 month ago"` |
| `git show <commit>` | Show commit details | `git show a73bb2` |
| `git reflog` | Show all reference logs | `git reflog` |
